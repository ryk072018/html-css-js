划分网格：使用px

划分网格：使用auto 和 %

划分网格：使用单位fr 表示把剩余空间平分，1fr表示占用1份

网格线：

        .boxFour {
          display: grid;
          grid-template-columns: 1fr 1fr 1fr;
          grid-template-rows: 1fr 1fr 1fr;
        }
        .boxFa {
          background-color: #80c063;
          grid-row-start: 1;
          grid-column-start: 1;
          grid-row-end: 3;
          grid-column-end: 3;
        }

网格线：（简写）grid-area:1/1/3/3

        .boxFive {
          display: grid;
          grid-template-columns: 1fr 1fr 1fr 1fr;
          grid-template-rows: 1fr 1fr 1fr 1fr;
        }
        .boxFia {
          background-color: #c08563;
          grid-area: 1/1/3/3;
        }

命名网格线:

          .boxSix {
            display: grid;
            grid-template-rows: [top] 1fr [middle] 1fr [bottom];
            grid-template-columns: [left] 100px [center] 1fr [right];
          }
          
          .boxSixa {
            background-color: #c149d1;
            grid-area: top/left/bottom/center;
          }

命名网格区域: 比如三行两列，grid-template-areas的值对应3行

          .boxSeven {
            display: grid;
            grid-template-columns: 200px 1fr;
            grid-template-rows: 80px 1fr 50px;
            grid-template-areas: "header header" "nav main" "footer footer";
          }
          
          .boxSevena {
            background-color: #d135e6;
            grid-area: header;
          }
        
          .boxSevend {
            background-color: #333;
            grid-area: footer;
          }
